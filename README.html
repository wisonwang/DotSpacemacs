我的spacemacs配置

参考： zilongshangren的spacemacs配置

使用

  1. 安装spacemacs

  2. 下载本项目，执行

```bash ln -s /project/path/.spacemacs /user/home/path/.spacemacs ln -s /project/path/private /user/home/path/.emacs.d/private

ln -s /project/path/.custom.el /user/home/path/.custom.el

emacs

```

.custom.el 包含和用户本地环境有关的配置所以要看情况添加修改
